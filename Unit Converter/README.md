# Unit Converter

A multi-tabbed unit converter built with Python and Tkinter. This tool allows for quick conversions across four major categories: Length, Weight, Temperature, and Currency.

<img src="unit_converter.png" width="500">

## Features

* **Real-time Conversion:** Results update instantly as you type or change units.
* **Tabbed Interface:** Clean separation between different conversion categories.
* **Categories:**
    * **Length:** Meters, Kilometers, Feet, Miles, Inches.
    * **Weight:** Kilograms, Grams, Pounds, Ounces.
    * **Temperature:** Celsius, Fahrenheit, Kelvin.
    * **Currency:** USD, EUR, GBP, INR, JPY.
## Requirements

* Python 3.x
* Tkinter (Included with standard Python installations)

## Installation

1.  Clone this repository or download the source code.
2.  Ensure you have Python installed.

Run the application using the following command:
